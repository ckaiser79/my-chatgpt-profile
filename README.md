# Simple Static Site

This project is a simple static website that utilizes Bootstrap for styling and jQuery for interactivity. The site is structured to be easily customizable and extendable.

## Project Structure

```
simple-static-site
├── index.html        # Main HTML document
├── css
│   └── styles.css    # Custom styles
├── js
│   └── main.js       # Custom JavaScript
├── .gitignore        # Git ignore file
├── LICENSE           # Licensing information
└── README.md         # Project documentation
```

## Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd simple-static-site
   ```

3. **Open `index.html` in your web browser** to view the site.

## Usage

- The site uses Bootstrap for responsive design. You can customize the styles in `css/styles.css`.
- Add your custom JavaScript functionality in `js/main.js`. This file is linked to the HTML and can utilize jQuery for enhanced interactivity.
