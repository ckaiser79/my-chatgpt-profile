Prompt for your GPT user profile

How to use this prompt

  1. Copy the prompt below using the "Copy Prompt" button.
  2. Open ChatGPT and start a new conversation.
  3. Paste the prompt into the chat.
  4. Copy the JSON output from ChatGPT's response in this page.
Analyze and create a comprehensive ChatGPT usage profile for 2025. Based on your knowledge of my interactions with you, provide:

1. An archetype name that describes my usage style (e.g., "Creative Coder", "Researcher", "Content Creator")
2. A brief summary of how I use AI in my work and projects
3. My interaction types - the main ways I use ChatGPT about 15 points for a wordcloud (e.g. brainstorming, coding, writing, research) with importance weights from 1-10
4. My experience level with AI on a scale of 0-10, including a brief reason for the rating
5. 2 actionable tips in 3 or 4 groups for improving my AI usage and prompting skills

The output langauge is defined in "header.outputLangeuage".

Return the response ONLY as valid JSON (no other text) matching this exact structure:

{

    },
    "archetype": {

    },
    "interactionTypes": [
        {
            "word": "brainstorming",
            "weight": 10
        },
        {
            "word": "coding",
            "weight": 9
        }
    ],
    "userLevel": {
        "value": 8,
        "reason": "I have extensive experience using AI for various coding and creative tasks, allowing me to efficiently integrate AI into my workflow."

        {
            "category": "Prompting",
